# placeHolder app

This application is deployed in the next url -> https://placeholder-app.netlify.app/

## Technologies
- React
- styled-components
- react-hot-toast
- react-hook-form
- react-slick
- react-icons

## Installation

Clone this repository with the next command

    git clone https://github.com/IrvingRR/placeholder-app.git

Access the folder 

    cd placeholder-app

Install dependencies

    npm install

Run application

    npm run dev
